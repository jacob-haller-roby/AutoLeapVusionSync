# AutoLeapVusionSync

# Requirements

Install nvm and node:

- `brew install nvm`
- `nvm install node`

# Install dependencies

At root of this package, run:

- `npm i`

# Update Sensitive Data

Copy the provided `.env.template` file to create a `.env` file.  Provide values for the variables included therein.

# Start Service

To start the program, at the root of this package, run:

- `npm start`

The service will continue to run as long as the terminal window is left open

# Stop Service

CTRL-C in terminal will exit the program
